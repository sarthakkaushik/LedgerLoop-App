# Expense Tracker Web

## Local run

```bash
npm install
npm run dev
```

Default URL: `http://localhost:5173`

The app points to backend `http://127.0.0.1:8000` by default.
Set `VITE_API_BASE_URL` in your frontend env for deployed environments.
Set `VITE_CLERK_PUBLISHABLE_KEY` to enable Clerk login in the auth screen.
